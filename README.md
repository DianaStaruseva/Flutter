ПРАКТИЧЕСКАЯ РАБОТА №1
Проектирование макетов 
Цель работы: используя виджеты фреймворка Flutter, разработать макеты страниц.
Ход работы:
1.	Страница Welcome
Для создания верстки страницы Welcome использовались такие виджеты как: Container,  Column, Text, OutlinedButton, TextButton, Image.
Для коррекции отображения использовались такие свойства виджетов как: padding, alignment, color, style, minimumSize, fontSize, backgroundColor, shape, borderRadius, textDirection, fontFamily.

 
2.	Страница Meditate
Для создания страницы Meditate использовались такие виджеты как: Container и его свойства decoration, alignment, child, padding, width, height;  виджеты Column и Row и их свойства textDirection, mainAxisAlignment, children;  Text и его свойства style, textAlign; виджет ElevatedButton.icon со свойствами icon, label, style и виджет  ImageIcon и его свойства color, size, image.
 

Вывод: в результате выполнения практического задания были изучены основные виджеты компоновки и оформления визуальной составляющей приложений.
